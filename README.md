## docker
### to install docker 
```
sudo apt-get remove docker docker-engine docker.io
sudo apt-get update
sudo apt-get upgrade -y
sudo apt install docker.io -y
sudo systemctl start docker
sudo systemctl enable docker
docker --version
sudo docker version
```
